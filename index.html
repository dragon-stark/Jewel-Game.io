Crystal Collector Game!

Game Rules


This game will begin by showing you a random number


Each time the game begins, the crystals below will be assigned random values.


To win you must reach the random number assigned without going beyond with the values from each crystal.


The value of each crystal is not given to you, part of the fun is finding the pattern that will match your number.


All values will reset at the start of a new game.
